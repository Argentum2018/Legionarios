Lineage 2 Chronicle 4 Administration Panel

What is it?

Lineage 2 Chronicle 4 Administration Panel (short named Lin2Panel) is panel for administration of Lineage 2 Chronicle 4 official files based servers. It is NOT written for L2J based servers, if you have one, use any of other administration panels for L2J.
